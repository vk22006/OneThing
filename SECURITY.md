# Security Policy

## Supported Versions

The following versions of **OneThing** are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.1.x  | :white_check_mark: |
| v1.0.x  | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you find any security vulnerabilities, please do **not** report them through public issues.

To report a vulnerability, please use the **GitHub Security Advisory** feature:
1. Go to the [Security tab](https://github.com/vk22006/OneThing/security) of the repository.
2. Click on **Advisories**.
3. Click on **New draft advisory**.

This will allow you to report the issue privately to the maintainers. We will investigate and respond as soon as possible.

Thank you for helping keep this project secure!
